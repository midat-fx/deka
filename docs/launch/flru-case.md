# FL.ru — кейс в портфолио / текст под отклики

Deka — сильный кейс для откликов на заказы по «Telegram-ботам», «ИИ/чат-ботам»,
«RAG», «автоматизации». Показывает не «сделал бота», а **инженерную зрелость**:
eval-гейт, честность модели, продакшн 24/7. Именно этого не хватает в портфолио
большинства фрилансеров, и это поднимает чек.

## Блок для портфолио (короткий)

**Deka — ИИ-ассистент по Налоговому кодексу РК-2026 (Telegram + веб)**

Telegram-бот, отвечающий на вопросы по налоговому законодательству **только
цитатами из первоисточника** (grounded RAG): галлюцинации цифр исключены по
построению, при неуверенности — честный отказ. Подбор налогового режима, трекер
оборота с алертами о лимитах, калькулятор формы 910, напоминания о дедлайнах,
3 языка (ru/kk/en).

Технически: собственный поиск по корпусу закона (стеммер + BM25 + векторный
поиск, слияние через RRF), **golden-set + eval-гейт в CI** (точность поиска
hit@3 = 94%, регрессии ловятся автоматически), продакшн на Cloudflare Workers +
D1 + Neon/pgvector, 24/7, приватность (хэш вместо id, вопросы не хранятся).

Стек: TypeScript, grammY, Cloudflare Workers/Pages/D1, Neon Postgres, Gemini.
Живой бот: @deka_tax_bot · Код: github.com/midat-fx/deka

> которые отвечают строго по вашим документам, а не «выдумывают». Свежий пример —
> Deka (@deka_tax_bot): ассистент по Налоговому кодексу с ответами по первоисточнику
> и цитатами, с автоматическим eval-гейтом качества в CI и продакшном на Cloudflare.
> Могу так же собрать бота по вашей базе знаний / регламентам / каталогу.
> Готов показать код и обсудить задачу.

## Почему это продаёт (не для публикации)

- «Grounded RAG + eval-гейт» — редкие слова в портфолио фрилансера; сразу выделяет.
- Живой продакшн-URL > 10 репозиториев без деплоя.
- Тема (налоги, точность, честность) читается как «умеет в ответственные домены».
- RAG-боты по документам — платёжеспособный спрос ($200–600+ за проект).
