## Почему сейчас

Форму 910 за 1-е полугодие сдают до **15 августа**. Это единственное окно, когда
ИП и самозанятые сами гуглят «как сдать 910», «ставка упрощёнки 2026», «лимит
самозанятого». За 2–3 недели до дедлайна спрос на помощь — пиковый. Значит:
**посев должен идти с ~25 июля по ~12 августа.**

## Правила посева в чатах (чтобы не забанили)

1. Сначала **ответь по делу** на чей-то вопрос, потом мягко: «кстати, собрал бота,
   который это считает — @deka_tax_bot». Не постить «в лоб» рекламу первым сообщением.
2. Один чат — один заход. Не дублировать в тот же день.
3. Всегда добавлять дисклеймер: «неофициальный помощник, сверяйтесь с КГД 1414».
4. Спрашивать разрешения у админов крупных чатов — многие ок с полезным ботом.

## Метрики успеха (смотреть в D1-телеметрии)

```
npx wrangler d1 execute deka-telemetry --remote \
  --command "SELECT event, COUNT(*) n, COUNT(DISTINCT user_hash) u FROM events GROUP BY event ORDER BY n DESC"
```

- `start` с payload `site910` / `site` → сколько пришло с лендинга
- `pro` / `waitlist` → сколько нажали «хочу Pro» (сигнал платить)
- `answer` = `answered` vs `cache_hit` → нагрузка и польза Q&A
- `intent` = `menu:*`, `nav:*` → чем пользуются
- DISTINCT `user_hash` по дням → живые пользователи (главная цифра для кейса)
