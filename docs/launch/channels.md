## 1. Ссылки с метками — по одной на канал

Бот записывает источник из deep-link (`?start=<метка>`), метка видна в
телеметрии. Латиница/цифры, без пробелов, до 32 символов. Используй СТРОГО
свою ссылку для каждого канала — иначе не поймём, что сработало.

| Канал | Ссылка |
|---|---|
| Тёплый круг (лично) | `https://t.me/deka_tax_bot?start=warm` |
| Мастера маркетплейса | `https://t.me/deka_tax_bot?start=masters` |
| Свой канал/статус | `https://t.me/deka_tax_bot?start=my` |
| TG-чаты ИП (посев) | `https://t.me/deka_tax_bot?start=chat` |

Смотреть результат (сколько пришло с каждого канала):

```
npx wrangler d1 execute deka-telemetry --remote --command \
  "SELECT detail AS source, COUNT(DISTINCT user_hash) AS users FROM events \
   WHERE event='start' AND detail IS NOT NULL GROUP BY detail ORDER BY users DESC"
```

## 2. Сценарии коротких видео (30–45 сек, запись экрана + голос)

Общее: вертикально, первые 2 секунды — хук, без вступлений «привет, я…».
В конце — метка-ссылка в описании и словами «бот Дека, ссылка в описании».
Дисклеймер в описании: «неофициальный помощник, сверяйтесь с КГД 1414».

**Видео 1 — «Ты уже попал на НДС?» (трекер)**
1. Хук (текст на экране): «Порог НДС в 2026 снизили В ДВА РАЗА. Проверь себя за 30 сек»
2. Экран: открываешь бота → пишешь «заработал 4 млн» → бот показывает прогресс-бары лимитов
3. Голос: «Пишешь боту доход обычными словами — он считает, насколько ты близко к порогу НДС, и предупредит заранее»
4. CTA: «Бесплатно, на казахском и русском. @deka_tax_bot»

**Видео 2 — «Форма 910 за 60 секунд»**
1. Хук: «До 15 августа. Не знаешь, что за форма 910 — смотри»
2. Экран: кнопка «📋 Форма 910» → сроки → калькулятор налога 4% от твоего оборота
3. Голос: «Кто сдаёт, когда, сколько платить — и расчёт по твоим цифрам, со ссылкой на статью кодекса»
4. CTA: «Успей до 15 августа. @deka_tax_bot»

**Видео 3 — «Спроси закон напрямую» (главная фишка)**
1. Хук: «ИИ выдумывает налоги. Этот бот — нет. Вот почему»
2. Экран: вопрос «какой лимит у самозанятого?» → ответ с плашкой «Ст. 718 п. 6» → тап по ссылке — открывается adilet.zan.kz
3. Голос: «Каждый ответ — из текста кодекса со ссылкой на статью. Не нашёл — честно скажет "не нашёл", а не придумает цифру»
4. CTA: «Проверь на своём вопросе. @deka_tax_bot»

> ссылками на статьи, считает форму 910 и следит за лимитами. Перед сдачей
> 910 (до 15 августа) вашей аудитории может быть полезно.
>
> Бот: https://t.me/deka_tax_bot?start=admin · сайт: deka-3st.pages.dev
> Он неофициальный и полностью бесплатный, ничего не продаю. Если посчитаете
> полезным — буду рад посту; могу прислать готовый текст. Спасибо!

## 4. Шаблон ЛС бухгалтеру

> Здравствуйте! Делаю бесплатного бота-помощника по НК РК-2026 для ИП и
> самозанятых (@deka_tax_bot — ссылка с меткой: t.me/deka_tax_bot?start=buh).
> Он снимает рутинные вопросы («какой лимит», «когда сдавать 910») ответами
> из текста кодекса со ссылками на статьи, а по сложным вопросам сам отправляет
> к бухгалтеру — он вам не конкурент, а фильтр однотипных вопросов.
>
> Если удобно — поделитесь с клиентами. Буду благодарен за обратную связь:
> чего боту не хватает, чтобы реально экономить ваше время?

## 5. Куда конкретно идти (собрать свой список)

Поиск в Telegram по словам: «ИП Казахстан», «упрощёнка КЗ», «самозанятые
Казахстан», «бухгалтер Алматы», «Kaspi магазин», «Wildberries Казахстан»,
«фрилансеры Казахстан». + городские чаты Алматы/Астаны. Заведи табличку:
чат → дата захода → реакция. Один чат = один заход, повтор не раньше чем
через неделю и только с новой пользой (например, «за 3 дня до дедлайна»).

## 6. Чего НЕ делать

- Не рассылать одно и то же сообщение веером в один день (теневой бан TG).
- Не спорить в чатах с бухгалтерами — соглашаться: «финальное слово за
  специалистом, бот — для ориентира».
- Не обещать того, чего нет («заменит бухгалтера», «одобрено КГД» — нельзя).
- Не покупать рекламу до того, как бесплатные каналы покажут отклик.
