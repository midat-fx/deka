## 1. Почему не просто ChatGPT (честно)
1. Свежие числа vs устаревшая память (НДС 16 vs 12) — сильный, но перишабельный ров (рычаг на август).
2. Принудительные цитаты со ссылкой до статьи (validateAnswer бракует ответ без ссылки) — GPT выдумывает номера.
3. Детерминированный визард/калькуляторы на константах, покрыты тестами — самое трудновоспроизводимое.
4. Statefulness: помнит ТВОЙ оборот и дедлайн, пингует за 7/1 день — GPT стейтлес.
5. Честная граница «нет в моих разделах → 1414» вместо блефа.
6. Fact-checker: Deka проверяет ответ GPT по первоисточнику.
